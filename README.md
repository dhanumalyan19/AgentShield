# AgentShield

Secure Execution Layer for Agentic AI using MCP.

## Project Overview

AgentShield is an AI-powered security platform that validates every action generated by an AI agent before execution.

The system uses:

- FastAPI
- Gemini API
- Model Context Protocol (MCP)
- MySQL
- SQLAlchemy
- JWT Authentication

The goal is to provide secure, auditable, and approval-based execution for enterprise AI agents.
